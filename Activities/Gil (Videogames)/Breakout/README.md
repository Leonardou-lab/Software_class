# Breakout
A01787221
Leonardo Andre Flores Mendoza

## Objective

Destroy all the bricks on screen by bouncing the ball with your paddle. Clear all 3 levels to win the game and don't let the ball fall, you only have 3 lives.

## Controls

| Key | Action |
|-----|--------|
| **A** | Move paddle left |
| **D** | Move paddle right |
| **Space** | Serve the ball and Restart after Game Over |

## Rules

- The ball bounces off the left, top, and right walls
- If the ball falls through the bottom you lose a life
- You start with 3 lives, when they run out it's Game Over
- Destroy all bricks in a level to advance to the next one
- The game has 3 levels, clearing all three wins the game
- Each level adds one more row of bricks (3, 4, 5 rows, 6 bricks per row)
- The total number of destroyed bricks is shown on screen at all times

## Powerups

Some bricks drop a powerup when destroyed. Let it fall and catch it with your paddle to activate it.

| Powerup | Color | Effect |
|----------|-------|--------|
| Extra Life | Red | Gain one additional life |
| Multiball | Cyan | Spawns two extra balls that also destroy bricks |
